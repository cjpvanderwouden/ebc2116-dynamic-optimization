# EBC2116 > Dynamic Modelling & Dynamic Optimization

Maastricht University | Department Quantitative Economics (2025–2026)

Pretty much all my coursework for EBC2116 (Prof. Timmermans) Dynamic Modeling & Optimization, including: compiled PDFs, handwritten derivations, and Python source code. Topics span finite/infinite-horizon optimal control, phase diagrams, complementary slackness, saddle-path stability, and Arrow's sufficiency theorem.

Use as reference, not a shortcut, Good luck :)
